# دورة الذكاء الاصطناعي · Intro to AI Course

موقع تعليمي تفاعلي بالعربية يُقدّم مقدمة شاملة إلى الذكاء الاصطناعي للمتعلّمين الجدد. كل المفاهيم مرفقة بنشاط تفاعلي يعمل بالكامل داخل المتصفّح بدون كاميرا أو رفع بيانات.

## الوحدات

1. مقدمة في الذكاء الاصطناعي
2. كيف يتعلم الذكاء الاصطناعي (مع مختبر شبكة عصبية)
3. الرؤية الحاسوبية (مع مصنّف أرقام)
4. النماذج اللغوية الكبيرة (مع مختبر مطالبات)
5. الوكلاء والذكاء الاصطناعي الوكيلي (مع باني وكيل)
6. الأنظمة الذاتية واتخاذ القرار
7. الأمن السيبراني والذكاء الاصطناعي (مع هجوم معادي)
8. الأخلاقيات والتحيّز + اختبار نهائي

## التشغيل محلياً

```bash
npm install
npm run dev
```

## البناء للنشر

```bash
npm run build
```

سيُولَّد المخرج في `dist/`.

## النشر على Netlify

### عبر CLI

```bash
npm i -g netlify-cli
netlify login
netlify deploy --build --prod
```

### عبر GitHub
ادفع المستودع إلى GitHub، اربطه من Netlify (يلتقط الإعدادات تلقائياً من `netlify.toml`).

## التقنيات
- React 19 + TypeScript + Vite
- Tailwind CSS مع دعم RTL
- Framer Motion، Lucide Icons، React Router
- جميع الأنشطة مكتوبة من الصفر بدون مكتبات ML خارجية
